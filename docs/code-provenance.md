# Code Provenance

This repository was assembled from the following original materials:

- a lab notebook export titled `Data Analysis Script and Output`
- a final thesis report PDF
- a rehearsal presentation PDF
- result figures exported during the project

## How To Read The Code

- `notebook_export/data-analysis-script-and-output.md` is the most complete original source and preserves the analysis notes and command history.
- `scripts/` contains curated standalone files reconstructed from that notebook export so reviewers can browse the code more easily.

## Important Context

- The project was executed in an HPC environment using Slurm job scripts and software modules.
- Absolute paths, usernames, and module names are preserved where they are part of the original workflow.
- Some scripts depend on input tables and intermediate files that are not included here.
- The absence of raw data does not affect the code examples as evidence of pipeline design, orchestration, and downstream analysis work.
