# figures/

This directory holds representative output figures generated by the R scripts in `scripts/r/`.

| File | Script | Description |
|---|---|---|
| `alpha_diversity.png` | `01_diversity_analysis.R` | Boxplots of Shannon H', Simpson 1-D, species richness, and Pielou evenness by land-use group |
| `beta_diversity_pcoa.png` | `02_ordination.R` | Bray-Curtis PCoA with 95% confidence ellipses; PERMANOVA annotation |
| `beta_diversity_nmds.png` | `02_ordination.R` | Bray-Curtis NMDS (stress < 0.15) |
| `differential_abundance_volcano.png` | `03_differential_abundance.R` | Volcano plot of DESeq2 genus-level results (agricultural vs. forest) |
| `differential_abundance_heatmap.png` | `03_differential_abundance.R` | VST-scaled heatmap of top 40 differentially abundant genera |
| `mag_completeness_contamination.png` | `04_mag_functional_heatmap.R` | Scatter plot of CheckM2 completeness vs. contamination for all MAGs |
| `functional_heatmap.png` | `04_mag_functional_heatmap.R` | Presence/absence heatmap of N/C cycling genes across medium/high-quality MAGs |

Raw data are not included in this repository.
Figure PNG files are generated by running the R scripts after providing the required input data.
